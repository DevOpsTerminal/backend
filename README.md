# DevOpsTerminal backend

DevOpsTermina Swagger UI Spec to existing Node.js/Express.js project

## Requirements

* Node.js v6.11.0
* MongoDB v3.4.6
* Gulp v3.9.1

## Usage

    npm i
    node server.js

Api documentation can be found on: `http://localhost:3000/api-docs`



## APICRA Scripts

    app_init.bat
    app_start.bat
    gulp_init.bat
    gulp_install.bat
